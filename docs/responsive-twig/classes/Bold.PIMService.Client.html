__construct(boolean|string $wsdl = false, array $soapConfig = array())

constructor for the client that interacts with the API. if you pass the wsdl URI and config settings, then the soapclient will be created in this constructor. If you dont do this, then you must manually set the SoapClient with the setSoapClient method before calling one of the API methods.
